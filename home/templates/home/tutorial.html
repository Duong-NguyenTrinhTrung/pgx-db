Search or Browse to any ATC code of interest

From Drugs/ATC tree browser, users can input a single ATC code or browse in the ATC tree to launch the search process. The ATC tree starts with 14 Anatomical groups and ends with chemical substance nodes. At any level, users can click on the rightmost button to show the network.
